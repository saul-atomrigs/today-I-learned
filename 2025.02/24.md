# <Visibility> 패턴
Modal, Dropdown, Accordion, Drawer, Toolip... 이들의 공통점은?

버튼 + 버튼을 누르면 show/hide 되는 UI의 합이라는 점

예를들어:
```ts
<Modal>
  <button>
  </button>
  <UI/>
<Modal>
```

이러한 형식을 일반화핧 수 있다:
```ts
<VisibilityProvider>
  <VisibilityTrigger>
    button
  </VisibilityTrigger>
  <VisibilityTarget>
    UI
  </VisibilityTarget>
</VisibilityProvider>
```
