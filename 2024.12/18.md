# Event Capturing, Event Bubbling

Capturing: DOM 트리 최상단에서 타깃 DOM 노드까지 아래로 이벤트가 내려오는 단계

Bubbling: 타겟 요소에서 이벤트 발생 후 다시 위로 올라가는 단계

event.stopPropagation으로 이벤트들의 상호 간섭을 방지
